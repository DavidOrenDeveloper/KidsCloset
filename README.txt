ארון הילדים - אפליקציית מעקב בגדים
=====================================

איך מריצים:
1. חלצו את קובץ ה-ZIP לתיקייה במחשב / בטלפון.
2. הכי טוב: העלו את כל הקבצים (כפי שהם, עם אותה מבנה תיקיות) לשירות אחסון פשוט
   כמו GitHub Pages / Netlify Drop / Vercel, ואז פתחו את הכתובת בדפדפן בטלפון.
   ככה תקבלו גם התקנה מלאה כ-PWA (הוספה למסך הבית) עם תמיכה אופליין.
3. אפשר גם פשוט לפתוח את index.html ישירות בדפדפן (לחיצה כפולה) - האפליקציה
   תעבוד באופן מלא (השמירה על המכשיר לא תלויה בשרת), אבל אייקון ה"התקנה למסך
   הבית" ותמיכת ה-Service Worker עשויים לא לפעול במצב פתיחה מקומי (file://)
   בחלק מהדפדפנים. באייפון אפשר תמיד להוסיף כל עמוד לדף הבית דרך כפתור השיתוף.

חשוב לדעת:
- כל המידע (ילדים, קטגוריות, פריטים) נשמר ב-localStorage של הדפדפן.
- כל התמונות נשמרות ב-IndexedDB של הדפדפן, בדחיסה אוטומטית (JPEG, עד 800px).
- מחיקת היסטוריית הגלישה / נתוני האתר בדפדפן תמחק גם את הנתונים האלה!
  לכן מומלץ לבצע "ייצוא גיבוי" מדי פעם דרך התפריט (☰) ולשמור את קובץ ה-JSON
  במקום בטוח (Drive, WhatsApp לעצמכם וכו').
- ייבוא קובץ גיבוי ידרוס את כל הנתונים הקיימים באפליקציה באותו מכשיר.
- אין שום שרת, מסד נתונים חיצוני, או שליחת מידע לאינטרנט - הכל נשאר במכשיר שלכם.

מבנה הקבצים:
index.html   - המבנה של האפליקציה
style.css    - העיצוב (מצב בהיר בלבד)
app.js       - כל הלוגיקה: שמירה, IndexedDB, סינון, ייצוא/ייבוא
manifest.json - קובץ ההגדרות של ה-PWA
sw.js        - Service Worker לשמירה אופליין של הקבצים הסטטיים
icons/       - אייקונים להתקנה במסך הבית
